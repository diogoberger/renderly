Renderly API Docs

Renderly converts HTML to PDF. POST your HTML, get a PDF back. Base URL: https://api.renderly.dev

Quickstart

1. Create a free account and copy your API key.
2. Make your first request:

curl -X POST https://api.renderly.dev/v1/pdf/from-html \
  -H "Authorization: Bearer rly_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"html": "<h1>Hello, Renderly!</h1>"}' \
  --output hello.pdf

Authentication

All API requests require a Bearer token in the Authorization header:

Authorization: Bearer rly_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxx

Generate and manage your keys in the dashboard.

POST /v1/pdf/from-html

Convert an HTML string to a PDF document. Returns application/pdf binary.

Request body

FieldTypeRequiredDescription
htmlstringrequiredFull HTML string to render
optionsobjectoptionalRendering options (see below)

Options object

FieldDefaultDescription
format"A4"Page size: A4, A3, A5, Letter, Legal, Tabloid
landscapefalseLandscape orientation
margin"1cm"Shorthand string or {top, right, bottom, left}
scale1Zoom scale (0.1–2)
printBackgroundtrueInclude CSS background colors/images
headerTemplate""HTML for page header (Starter+ plan)
footerTemplate""HTML for page footer (Starter+ plan)
// Full example with options
{
  "html": "<html><body><h1>Q1 Report</h1></body></html>",
  "options": {
    "format": "Letter",
    "landscape": false,
    "margin": { "top": "2cm", "right": "1.5cm", "bottom": "2cm", "left": "1.5cm" },
    "footerTemplate": "<div style='font-size:10px;text-align:center'>Page <span class='pageNumber'></span> of <span class='totalPages'></span></div>"
  }
}

POST /v1/pdf/from-url

Navigate to a live URL and render it to PDF. Useful for capturing your own app's pages.

FieldTypeRequiredDescription
urlstringrequiredFully-qualified URL (https://...)
optionsobjectoptionalSame options as from-html
curl -X POST https://api.renderly.dev/v1/pdf/from-url \
  -H "Authorization: Bearer rly_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com", "options": {"format": "A4"}}' \
  --output page.pdf

API Keys

POST /v1/keys — Create a new API key
GET /v1/keys — List all keys (prefixes only)
DELETE /v1/keys/:id — Revoke a key

GET /v1/health

Public health check. Returns 200 if the API is operational, 503 if degraded.

Node.js example

const fs = require('fs');

const res = await fetch('https://api.renderly.dev/v1/pdf/from-html', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer rly_live_YOUR_KEY',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    html: '<h1>Invoice #42</h1><p>Amount: $1,200</p>',
    options: { format: 'A4', margin: '1.5cm' },
  }),
});

const buffer = await res.arrayBuffer();
fs.writeFileSync('invoice.pdf', Buffer.from(buffer));
console.log('Saved invoice.pdf');

Python example

import requests

response = requests.post(
    'https://api.renderly.dev/v1/pdf/from-html',
    headers={
        'Authorization': 'Bearer rly_live_YOUR_KEY',
        'Content-Type': 'application/json',
    },
    json={
        'html': '<h1>Invoice #42</h1><p>Amount: $1,200</p>',
        'options': {'format': 'A4', 'margin': '1.5cm'},
    },
)

with open('invoice.pdf', 'wb') as f:
    f.write(response.content)

print('Saved invoice.pdf')

PHP example

$ch = curl_init('https://api.renderly.dev/v1/pdf/from-html');
curl_setopt_array($ch, [
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer rly_live_YOUR_KEY',
        'Content-Type: application/json',
    ],
    CURLOPT_POSTFIELDS => json_encode([
        'html' => '<h1>Invoice #42</h1>',
        'options' => ['format' => 'A4'],
    ]),
    CURLOPT_RETURNTRANSFER => true,
]);
file_put_contents('invoice.pdf', curl_exec($ch));
curl_close($ch);
